## Todos
Pet project. Simple task manager.

Author: Yuri Shvetsov

Created date: 01/05/21

------------
Technologies

- Vite
- Vue 3
- Vuex

------------

``` zsh
# Install dependencies:
npm i

# Server with hot reload at http://localhost:5173/
npm run dev

# Output will be at dist directory
npm run build

# Preview project
npm run preview
```
