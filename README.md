Space Lite
==========

This is a simple todolist system implemented with Vue.

## Features

- Folders
- Task editing
- Hidden tasks
- Export/import to json
- Dark theme


## Build setup

Install dependencies
``` zsh
npm i
```

Server with hot reload at http://localhost:5173/
```
npm run dev
```

Output will be at dist directory
```
npm run build
```

Preview project
```
npm run preview
```
