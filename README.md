## Todolist-vue
Learning project on JavaScript and Vue.js

Author: Yuri Shvetsov

Date: 01/05/21

------------
Objectives:
- learning of **Vue** framework of version 3
- at using Vuex

------------

``` zsh
# Install dependencies:
npm i

# Server with hot reload at http://localhost:9000/
npm start

# Output will be at dist directory
npm run build
```
